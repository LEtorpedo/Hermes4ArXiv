## 🚀 快速开始

**第一次使用？** 只需3分钟：
1. 参考 [主README](../README.md) 了解基本流程
2. 按照 [DeepSeek配置指南](setup/DEEPSEEK_SETUP_GUIDE.md) 获取API密钥
3. 参考 [Gmail设置指南](setup/GMAIL_SETUP_GUIDE.md) 配置邮件授权码
4. 在GitHub Secrets中添加配置，启用Actions即可！
