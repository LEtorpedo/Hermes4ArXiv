## 🆘 常见问题

**邮件发送失败？**
- 检查Gmail应用密码是否正确
- 确认邮箱开启了SMTP服务

**AI分析失败？**  
- 检查DeepSeek API密钥和余额
- 确认网络连接正常

**没有收到邮件？**
- 检查垃圾邮件文件夹
- 确认收件人邮箱地址正确
